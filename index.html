Guía de Estudio de Cálculo Diferencial

Dominio y Rangos

El dominio de una función es el conjunto de valores para los cuales la función está definida. Los rangos, por otro lado, son los valores que la función puede tomar.

A continuación se muestra un ejemplo de cómo determinar el dominio y los rangos de una función:

// Ejemplo de función

function f(x) {

    return x * x;

}

// Paso 1: Determinar el dominio

// El dominio de la función f(x) = x * x es (-∞, ∞) porque está definida para cualquier número real.

// Paso 2: Determinar el rango

// El rango de la función f(x) = x * x es [0, ∞) porque el resultado siempre es igual o mayor que cero.

Límites de Cálculo Diferencial

Los límites son fundamentales en el cálculo diferencial. Representan el comportamiento de una función cuando se acerca a un determinado valor.

A continuación se muestra un ejemplo de cómo calcular un límite:

// Ejemplo de límite

lim(x -> 2) (x^2 + 3x - 2) = 8

// Paso 1: Sustituir el valor de x en la función

// (2^2 + 3*2 - 2)

// Paso 2: Calcular el resultado

// (4 + 6 - 2) = 8

Composición de Funciones

La composición de funciones es una operación en la que se aplica una función sobre otra función.

A continuación se muestra un ejemplo de cómo componer funciones:

Factorizar

Factorizar una expresión algebraica significa escribirla como un producto de factores más simples.

A continuación se muestra un ejemplo de cómo factorizar una expresión:

// Ejemplo de factorización

x^2 + 5x + 6 = (x + 2)(x + 3)

// Paso 1: Buscar dos números que sumen 5 y multipliquen 6

// Paso 2: Descomponer el término lineal

// x^2 + 2x + 3x + 6

// Paso 3: Agrupar los términos en común

// (x^2 + 2x) + (3x + 6)

// Paso 4: Factorizar cada grupo

// x(x + 2) + 3(x + 2)

// Paso 5: Extraer el factor común

// (x + 2)(x + 3)

Derivar por 4 Pasos

La derivación es una operación fundamental en el cálculo diferencial. Se utiliza para encontrar la tasa de cambio instantánea de una función en un punto dado.

A continuación se muestra un ejemplo de cómo derivar una función utilizando el método de los 4 pasos:

// Ejemplo de derivación

// Paso 1: Encontrar la función derivada de cada término

2x^3 = 6x^2

3x^2 = 6x

// Paso 2: Simplificar

6x^2 + 6x

// Paso 3: Combinar términos similares

6x^2 + 6x

// Paso 4: Obtener la función derivada final

f'(x) = 6x^2 + 6x
